Installing the latest version of Redis on Ubuntu

// under Python Django

Had trouble this morning restarting my Vagrant development environment. I was getting a Redis wrongresponse error on my Django application (wrong number of commands set). A bit of Googling revealed that the problem was related to an outdated version of Redis (2.1.4).

I tried sudo apt-get install redis-server, but apparently this doesn't fetch the latest repository from ppa.

The commands required to upgrade redis-server to the latest version (thanks this post):

$ sudo apt-get install -y python-software-properties
$ sudo add-apt-repository -y ppa:rwky/redis
$ sudo apt-get update
$ sudo apt-get install -y redis-server

As explained in the previously hyperlinked post,

Line 1 – So we want to add a new repository that contains the latest version of Redis, the easiest way to do this requires add-apt-repository, which most likely isn’t installed on your machine, so the first thing we want to do is install python-software-properties, this gives us access to add-apt-repository.

Line 2- Next we want to set up this new repository.

Line 3 – Now we want to update apt so it sees any changes we have made.

Line 4 – Finally we want to install redis-server, now the latest version will be installed.

The -y flag is useful when using these steps in a provisioning tool such as Ansible.
